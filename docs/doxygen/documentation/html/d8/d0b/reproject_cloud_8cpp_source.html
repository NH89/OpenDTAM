13 // Camera Matrix modification to reproject with inverse depth:
14 // Recall that the real camera matrix looks like:
15 // [ f 0 cx 0] [ xc ] [ xp*zc] [xp]
16 // [ 0 f cy 0]*[ yc ]=[ yp*zc]=[yp]
17 // [ ? ? ? ?] [ zc ] [ ? ] [? ]
18 // [ 0 0 1 0] [ wc ] [ zc ] [1 ]
19 //
20 // Notice that the z term of the output has been lost, so we can fill
21 // it in with whatever we want. To make it invertable and simple let's
22 // do this:
23 //
24 // [ f 0 cx 0] [ xc ] [ xp*zc] [ xp ]
25 // [ 0 f cy 0]*[ yc ]=[ yp*zc]=[ yp ]
26 // [ 0 0 0 1] [ zc ] [ 1 ] [1/zc ]<--inverse depth!
27 // [ 0 0 1 0] [ 1 ] [ zc ] [ 1 ]
28 //
29 // How Convienient! We got inverse depth for free!
30 //
31 // Now lets look at how this matrix is used to reproject inverse depth to
32 // another image(for compactness I'll write the elements of the camera matrix
33 // by index:
34 //
35 // [0 1 2 3] [ ] [0 1 2 3]-1 [xp ] [x']
36 // [4 5 6 7]*[ A ]*[4 5 6 7] *[yp ]=[y']
37 // [8 9 a b] [ ] [8 9 a b] [1/zc] [z']
38 // [c d e f] [ ] [c d e f] [1 ] [w']
39 //
40 // Now we want the camera matrix to look nice, so swap the bottom
41 // two rows:
42 // [0 1 2 3] [ ] [0 1 2 3]-1 [xp ] [x']
43 // [4 5 6 7]*[ A ]*[4 5 6 7] *[yp ]=[y']
44 // [c d e f] [ ] [c d e f] [1 ] [w']
45 // [8 9 a b] [ ] [8 9 a b] [1/zc] [z']
46 //
47 // Now we have:
48 // [ f*s 0 (cx-.5)*s+.5 0] [ xc ] [ xp*zc] [ xp ]
49 // [ 0 f*s (cy-.5)*s+.5 0]*[ yc ]=[ yp*zc]=[ yp ]
50 // [ 0 0 1 0] [ wc ] [ zc ] [ 1 ]
51 // [ 0 0 0 1] [ zc ] [ 1 ] [1/zc ]
52 //
53 // But wait, now the output is getting divided by the wrong number,
54 // and the input has a 1/zc in the w slot, which is not allowed
55 // (only a 1 is allowed there).
56 // Well, overall, we have something of the form:
57 // [0 1 2 3] [xp ] [x']
58 // [4 5 6 7]*[yp ]=[y']
59 // [8 9 a b] [1 ] [w']
60 // [c d e f] [1/zc] [z']
61 //
62 // So swap the last two rows and last two columns:
63 // [0 1 3 2] [xp ] [x']
64 // [4 5 7 6]*[yp ]=[y']
65 // [c d f e] [1/zc] [z']
66 // [8 9 b a] [1 ] [w']
67 //
68 // But we really don't care about the z of the output, so:
69 // [0 1 3 2] [xp ] [x']
70 // [4 5 7 6]*[yp ]=[y']
71 // [8 9 b a] [1/zc] [w']
72 // [1 ]
73 // So all we really needed to do is swap the two last columns
74 // and drop the last row.
75 // Thus ends the derivation.
