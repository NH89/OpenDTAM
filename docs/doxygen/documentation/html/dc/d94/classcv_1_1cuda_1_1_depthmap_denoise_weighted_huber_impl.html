Detailed Description

OpenDTAM Variant of Chambolle & Pock denoising

The complicated half of the DTAM algorithm's mapping core, but can be used independently to refine depthmaps.

Contributed by Paul Foster for GSoC 2014 Original algorithm described by Richard Newcombe, Steven J. Lovegrove, and Andrew J. Davison. "DTAM: Dense tracking and mapping in real-time."
